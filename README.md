# FlowScope Website

Official website for [FlowScope](https://pub.dev/packages/flowscope) — a Flutter in-app debugging SDK.

**Live at:** [flowscope.dev](https://flowscope.dev)

---

## About

This is the landing page for FlowScope, a Flutter debugging SDK that gives developers real-time visibility into Riverpod state, network requests, and events from an in-app overlay.

---

## Stack

Plain HTML + Tailwind CSS (CDN). No build step, no framework, no dependencies.

---

## Running Locally

Just open `index.html` in your browser. No server needed.

Or with a local server:
```bash
npx serve .
```
